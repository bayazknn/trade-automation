Command documentation is in `docs/freqtrade/commands/`.

- `freqtrade backtesting`: Run backtesting

- `freqtrade list-pairs`: List pairs
- `freqtrade list-exchanges`: List exchanges
- `freqtrade new-strategy`: Create new strategy
- `freqtrade plot-dataframe`: Plot dataframe
